# Руководство пользователя системы учета посещаемости

## 1. Назначение системы
Система предназначена для:
- фиксации посещаемости по классам за конкретную дату;
- ведения списка отсутствующих с причинами;
- просмотра статистики за дату;
- экспорта статистики в `Excel` и `CSV`.

Система поддерживает 2 роли:
- `admin` (администратор);
- `teacher` (учитель).

## 2. Вход в систему
1. Откройте веб-интерфейс.
2. В поле `API Base URL` укажите адрес API (по умолчанию используется адрес из адресной строки. Ручное заполнение необходимо лишь в отдельных случаях).
3. Введите логин и пароль.
4. Нажмите `Войти`.

После входа в верхней части экрана отображается роль и `userId`.

## 3. Навигация по интерфейсу
Основные вкладки:
- `Классы` (только для `admin`);
- `Посещаемость`;
- `Статистика`;
- `Учителя` (только для `admin`).

Дополнительно:
- кнопка `Выйти` завершает сессию;
- кнопка `Тёмная тема / Светлая тема` переключает оформление интерфейса.

## 4. Работа учителя

### 4.1. Заполнение посещаемости
Вкладка: `Посещаемость` → блок `Заполнить / обновить посещаемость`.

1. Выберите дату.
2. Выберите класс.
3. Дождитесь автозагрузки данных.
4. Укажите:
   - `По списку` (`totalStudents`);
   - `Присутствуют` (`presentCount`).
5. Добавьте отсутствующих:
   - `Отсутствующие по неуважительным причинам`;
   - `Отсутствующие по уважительным причинам` (с обязательной причиной - болезнь/официальное мероприятие(олимпиада, соревнование и т.д)/семейные обстоятельства/другое).
6. Нажмите `Сохранить посещаемость`.

Валидация:
- `presentCount` не может быть больше `totalStudents`;
- количество отсутствующих должно равняться `totalStudents - presentCount`;
- дубли фамилий не допускаются. Если это необходимо, добавьте имя.
- для уважительной причины поле причина обязательна.

### 4.2. Просмотр посещаемости
Вкладка: `Посещаемость` → блок `Просмотр посещаемости за дату`.

1. Выберите дату.
2. Выберите класс.
3. Нажмите `Загрузить`.

### 4.3. Просмотр статистики
Вкладка: `Статистика`.

1. Выберите дату.
2. Выберите класс (или `Все классы`).
3. Нажмите `Показать`.

Результат:
- список отсутствующих;
- класс и причина отсутствия;
- общее число отсутствующих.

### 4.4. Экспорт статистики
Вкладка: `Статистика`.

1. Выберите дату и (при необходимости) класс.
2. Нажмите:
   - `Экспорт в Excel` для `.xlsx`;
   - `Экспорт в CSV` для `.csv`.

Файл скачивается автоматически. Иногда формирование таблицы может занимать некоторое время в силу некоторых причин.

## 5. Работа администратора

### 5.1. Управление классами
Вкладка: `Классы`.

Функции:
- выбор класса;
- создание класса;
- просмотр текущего учителя выбранного класса;
- переназначение класса другому учителю;
- удаление класса.


### 5.2. Управление учителями и пользователями
Вкладка: `Учителя`.

Доступные операции:
- регистрация учителя;
- изменение учётных данных учителя;
- изменение собственных учётных данных;
- смена роли пользователя (`teacher`/`admin`).

Ограничение безопасности:
- назначенный администратор не может изменить роль админа, который его назначил.

### 5.3. Незаполненные классы
Вкладка: `Статистика`.

1. Выберите дату.
2. Нажмите `Показать незаполненные классы`.

Для `admin` выводится таблица:
- класс;
- учитель;
- `teacherId`.

## 6. Рекомендации по работе
- Сначала выбирайте дату и класс, затем вводите данные.
- При необходимости спользуйте экспорт для удобной работы с данными в Excel таблице.
- При большом числе классов используйте режим `Все классы` для быстрого контроля.

## 7. Частые проблемы

### 7.1. Не удаётся войти
Проверьте:
- правильность `API Base URL`(если его требовалось менять);
- логин/пароль;
- доступность backend.

### 7.2. Ошибка при сохранении посещаемости
Проверьте:
- соответствие формулы `totalStudents - presentCount`(соответствие количества введённых отсутствующих с ожидаемым количеством);
- отсутствие дублей в списке отсутствующих;
- заполнены ли причины для отсутствий по уважительным причинам.

### 7.3. Не скачивается файл экспорта
Проверьте:
- выбрана ли дата;
- не блокирует ли браузер загрузки.
- подождите несколько минут, формирование таблицы занимает некоторое время
